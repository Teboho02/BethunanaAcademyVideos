Place the Android APK here as:

    bethunana-academy.apk

Download it from the latest EAS preview build
(https://expo.dev/accounts/teboho02/projects/BethunanaAcademy/builds),
rename it to bethunana-academy.apk, and drop it in this folder.

The "Download for Android" button on the site links to /app/bethunana-academy.apk
(see src/app/constants/appDownload.ts). Replace this file on each new release.
